									********** Release Document for the NBA-Predictions Project v1.0-Beta **********


Note: This release is a beta-version. 

This is the first official release for the NBA-Predictions project. This release includes the Team-Prediction script which can be used to predict the score of upcoming NBA games. 

To use the scripts in this release ensure that your environment has the requisite packages outlined in each script's documentation. As of this release, the requisite packages are: 

	1. pandas
	2. seaborn
	3. TensorFlow
	4. time
	5. nba_api
	6. math

The documentation for the following scripts can be found at the corresponding locations
	
	Team-Prediction: https://github.com/RohanPankaj/NBA-Prediction/blob/master/docs/Team-Prediction.md
